# Advent of Code 2022

Solutions for Advent of Code 2022 in Rust. The solution runner relies on
[rudolf-rs](https://github.com/wallabythree/rudolf-rs) to download and cache
puzzle inputs.

## Main branch

Regular solutions.

## Optimised-for-speed

Solutions optimised for speed.
